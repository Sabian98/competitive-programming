A - Podmo Pukur
Ranklist
Time Limit: 2 second(s) Memory Limit: 32 MB

A water lily doubles itself in size each day in Podmopukur. From the time its first leaf appeared to the time when the surface of the pond was completely covered took n days. Now it is your task to calculate how long did it take for the pond to be 1/m covered?

Input

Input starts with an integer T (T<=50), denoting the number of test cases. Each case contains a single line with one integer n (1<=n<=10^9 ) and a fraction in 1/m form. You can safely assume that m is a power of 2 and less than 2^30 .For clarity see the sample input.

Output

For each case, print the case number and a integer which is the number of days it will take for the pond to be 1/m covered .If it is not possible to calculate print “Impossible” without the quotes.

Sample Input Sample Output
2
1 1/2
80 1/16
 Case #1 : Impossible
Case #2 : 76
